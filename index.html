Description

This dataset contains transaction records with features like transaction amount, location, time, merchant information, and fraud labels. Perfect for building fraud detection models, anomaly detection, and financial security applications.

Transaction Records

Comprehensive transaction data with detailed information including amounts, timestamps, and merchant details.

User Behavior Features

Features capturing user behavior patterns including transaction frequency, account age, and transaction history.

Fraud Labels

Binary labels (fraud/normal) for supervised learning and model evaluation.

Imbalanced Dataset

Realistic imbalanced dataset (5% fraud) for practicing imbalanced classification techniques.

Ready for Classification Models

Preprocessed and structured data ready for machine learning models including Random Forest, XGBoost, and Neural Networks.
